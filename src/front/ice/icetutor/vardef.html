Ingres Web Deployment Option allows developers to embed variables in their web pages, and use these variables in other pages. Page: Variables can only be used in the current page. There are two ways to set the value of this type of variable.
  • By including name/value pairs at the end of the URL string.
    For example, "/foo.html?name=john". This will allow the developer to use the variable "name" in the page "foo.html".
  • Using a macro command. An explanation follows.
