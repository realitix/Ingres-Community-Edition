To obtain the value of a variable, you only need to prefix the variable name with a colon.

For example, as already shown, a variable can be set with the value of another variable.

<!-- #ICE DECLARE=`session.var=:page` -->

A variable can be displayed using the "VAR" keyword.

<!-- #ICE VAR=`:page` -->

Variables can be used anywhere in any kind of Web Deployment Option macro statement.
