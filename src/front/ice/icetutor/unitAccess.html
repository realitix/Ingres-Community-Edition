The unit owner must declare how users will access the unit. Using the Web Deployment Option administration tool, three types of access can be granted.

Visible:
a role or user can see project information but cannot view the actual documents.
In practice, this level of access allows roles or users to see the complete list of documents.
Read:
a role or user can view all documents.
Create:
a role or user is authorized to create a new document in this unit.
