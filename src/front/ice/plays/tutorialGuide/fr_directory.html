Create Directory

This application will come under Web Deployment Option security and will not be visible to the HTTP server. We will therefore not create a directory under your web root directory for this application; rather we will choose one elsewhere in the file system. Later, we will register that location with the ICE server, which will then make the location visible through the HTTP server.
UNIX
cd my/ice/root/directory (for example: /usr/web/documents)
mkdir plays
cd plays
Windows32
cd my\ICE\root\directory (for example: c:\)
mkdir plays
cd plays
