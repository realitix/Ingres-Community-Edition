Create Skeleton HTML Files

To save time, we will create all the files we will need and then, register them with the ICE server.

Either use the example files that you received on your distribution or, using your favorite HTML editor, create the following template in the directory you created for the purpose:

   
<!DOCTYPE HTML PUBLIC "-//W3C/DTD HTML 4.0//EN">
<html>
<head>
<META HTTP-EQUIV="Content-Style-Type" CONTENT="text/css">
<LINK HREF="<!-- #ICE INCLUDE=`plays[play_styleSheet.css]` -->" TYPE="text/css" REL="STYLESHEET">
<title>Title of page</title>
</head>
<body>
<h1>Title of page</h1>
</body>
</html>

The line

   
<LINK HREF="<!-- #ICE INCLUDE=`plays[play_styleSheet.css]` -->" TYPE="text/css" REL="STYLESHEET">

includes a reference to the style sheet that determines the styles that apply to the various elements in the tutorial.

Style sheets help us to separate appearance or style from the information or content of our website. By including the style sheet using this technique, we are able to bring the style, in addition to the content, of our site under Web Deployment Option control. In this tutorial we use level 1 of the cascading style sheet mechanism, recommended by the W3C at http://www.w3.org/TR/REC-CSS1
