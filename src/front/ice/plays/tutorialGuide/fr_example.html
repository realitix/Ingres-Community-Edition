Example List

This tutorial is made up of a number of worked examples.

We start using simple concepts of HTML and Web Deployment Option to select rows from a single table. We then examine the various ways in which we can present a subset of the table rows using an HTML selector (combo box), hyperlinks and image based hyperlinks. We finish with a shopping application that involves more advanced concepts including joins and transaction control (commit/rollback/auto-rollback). For example, if a user terminates the connection, the contents of their shopping bag are removed automatically.

The examples show:

  • Simple select
  • Select, wraps back to row 1
  • Selector subset
  • Text hyperlink subset
  • Graphical hyperlink subset
  • Graphical hyperlink using a switch
  • Internet Commerce application
