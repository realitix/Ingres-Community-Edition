//////////////////////////////PREPROSESSOR DEFINITION/////////////////////////
******************************************************************************
There is simulated data configuration using the defines:
    SIMULATION    : simulated data
    FGCOLOR       : use foreground text color instead of background text color 
                    for the list control of transactions.
    SORT_INDICATOR: set the bitmap in the list control header 
                    to indicate the sort direction.
