Ingres Ruby Frequent Flyer Demo

This archive/directory structure contains a sample demonstration application that shows how to connect and query an Ingres Database using the Ingres Ruby driver.

Installation

Information on obtaining a copy of this demonstration application as well as the requirements and install process can be found at community.actian.com/wiki/Ruby/Learn/Flight_Planner_Demo_Application.
