Routes Form

The Routes form requires four data items for both the departure and arrival airport to retrieve the flights result set. These four data items include:

  • Country
  • Region
  • Airport
  • Day

The following three parameters are used to filter data from the database:

  • Departing airport code
  • Arriving airport code
  • Operating days

The Return Journey parameter extends queries passed to Ingres with the same criteria as for the outward bound queries, but with the departing and arriving airport codes swapped.

The list of airports is reduced by restricting the available airports based on the region. The list of regions is reduced by restricting the possible regions based on the country.
