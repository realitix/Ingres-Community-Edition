Query Execution

Clicking the Go button on the Routes form triggers the execution of the routeGoButton_Click event handler. This event handler performs the following actions:

  • Extracts the required parameter values from the Windows Form controls.
  • Initializes the DataGridView control that will receive the query results.
  • Binds the DataSet to the DataGridView.
  • Executes the SELECT query that populates the DataSet.

The results of the query are bound to the DataGridView, and then displayed.
