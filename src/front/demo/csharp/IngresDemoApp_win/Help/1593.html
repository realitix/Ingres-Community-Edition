Define Connection String

You must configure a connection string before connecting to a database using a .NET application.

To define a connection string

Enter the connection information using the Connection String Editor of the Ingres Data Adapter Configuration Wizard:

The wizard generates the following connection string:

Host=(local);Port=II7;Database=demodb

Note: User ID and password are optional when connecting to a local Data Access Server and the current logged-in user is also a valid Ingres user.
