How You Load the Country Combo Box Control

The values in the Departing and Arriving Country combo boxes are returned by a query from the database using an IngresDataAdapter control.

Note: The following procedure assumes that a combo box has been placed on the Visual Studio design form.

To load the Country combo box control:

  1. Add an IngresDataAdapter control.
  2. Construct the query.
  3. Generate a DataSet.
  4. Bind the combo box control to the DataSet.
  5. Populate the DataSet.
