Welcome

Welcome to the "Frequent Flyer" demonstration application, which is designed to introduce you to developing applications with Ingres, C#, Visual Studio 2005, and the .NET Data Provider 2.0.

The application consists of the following:

  • A database supporting NFC Unicode data called "demodb" created when Ingres or the demonstration is installed.
  • Sample data related to air travel—for example, airlines, airports, and routes.
  • Sample code illustrating common functionality that an application developer may want to write.

We hope that you find this application useful. If you have questions or comments, please raise an issue with Ingres Technical Support at http://servicedesk.actian.com.

Note: Knowledge of C# and Visual Studio is necessary to understand the examples.
