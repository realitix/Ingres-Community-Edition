Welcome to the Ingres PHP IFF Demo

Environment Requirements

To run the Ingres Demo you need the following components
  • Apache Webserver (2.x or higher)
  • PHP 5.x (or higher)
  • Ingres DBMS (9.1.0 or higher)
  • Ingres PHP driver 1.4.3 (available on http://pecl.php.net/package/ingres).
    If you need more help on installing and configuring the PHP driver please see http://community.actian.com/wiki/PHP_Driver

Unpacking the tarball

To unpack the tarball just execute the command: 
	tar -xvzf ingresPhpIff.tgz
This will create an subdirectory called IFF including all necessary files.

Configuring the IFF application

The configuration of the IFF is straight-forward. Just open the editor of your choice and point it to the file LocalSettings.php that resides within the just created directory. You'll find three constants in there: 
define('DATABASE','demodb');
define('USER','ingres');
define('PASSWORD','');
Please fill in the necessary information. This constants are used for the default connection to the Database. You can change the connection information within the IFF too. But that's only for the lifetime of the PHP Session.

If you want the application to work after you point your browser to the file "index.php" in the directory where you just unpacked the application then you must fill in the correct values.

After you changed the connection information within the application and you want to use the default credentials, just type "(local)" in the field named "hostname" and nothing else.
